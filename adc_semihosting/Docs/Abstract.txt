Project:
adc_semihosting

Hardware Requirements:
LPCXpresso Board LPC17xx

Project Objective:
Read ADC value and display bar graph using semi-hosting.

Project Details:
As this project only has semi-hosted output, it is effectively
useless in a "Release" context. Thus the only configuration for
the project is "Debug"
